# Descripción
Este proyecto consiste en un sistema de consola desarrollado en C# para administrar personas de una organización solidaria.
El sistema permite registrar voluntarios y coordinadores, almacenar su información y mostrar los datos registrados.

# Funcionalidades
- Registrar voluntarios.
- Registrar coordinadores.
- Mostrar listado de personas registradas.
- Mostrar horas trabajadas de los voluntarios.
- Mostrar área asignada y cantidad de personas a cargo de los coordinadores.

# Trabajo colaborativo con Git y GitHub
Para el desarrollo se utilizó un repositorio compartido en GitHub.
* Se realizaron las siguientes acciones:
- Creación y clonación del repositorio.
- Creación de ramas individuales para cada integrante.
- Desarrollo de clases por separado.
- Commits de los cambios realizados.
- Push de las ramas al repositorio remoto.
- Pull para actualizar cambios.
- Merge de las ramas al proyecto principal.
De esta manera se aplicó el trabajo colaborativo utilizando Git y GitHub.
